# TheqooSummarizer
Collect articles from Theqoo, translate into Vietnamese and group common comments.

### Lib
```
pip install bs4 openai seleniumbase python-dotenv
```

### Setup environment
- Create .env file
- Add config OPEN_AI_KEY

### How to run
```
.../summarier.py --id ID
```
Example:
https://theqoo.net/hot/3494143968
=> ID = 3494143968
